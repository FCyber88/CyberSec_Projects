## About This Portfolio
This repository contains a collection of **network traffic analysis and malware PCAP investigations** conducted as part of hands-on cybersecurity projects. Each case study includes:

- Metadata (analyst, date, tools, case file)  
- Investigation steps using Wireshark and other tools  
- Indicators of Compromise (IOCs)  
- Conclusions and findings  

These projects demonstrate practical skills in **threat detection, malware analysis, and SOC operations**.
